[ COMMENT from siromaru ]
BOFは以前2004、2005、2008、2010に参加しました。
2010が終わった後、今更のようにふと気づいたんです。
上記すべて「BOFのために曲作ってないなぁ」と…。

そう、いつもその直前にたまたま作ってた曲をBMS化した、
みたいなパターンばっかりだったんです。
別に書き下ろしたからといってクオリティや実力が変わるだけでもないんですが、
なんというか"BOF優勝目指そうぜ"感が全然足りてなかったと思います。気持ちが弱いというか。

ということで！BOF2011は、精一杯やっていこうと決心しました。
念頭に置いたのは「プレイヤーが楽しめること」。コレに尽きます。
より多くの人をより深く楽しませれば結果は自ずと付いてくるはず！をモットーに。
そしてそれはどうしたらいいか…。

考えた結論として出た答えの1つが、crankyさんとの合作でした。
crankyさんの超カッコイイメロディーに自分の下地を合わせればとんでもないモンが
できるんじゃないかと思い…。結果はもうまさに、siromaru + crankyという
重いリズムを刻みながらもメロディアスで展開のある良い楽曲に仕上がったと思います。

そしてiimo君の素敵ムービー。
従来の僕の曲向けの機械的なのをベースにしつつ、crankyさん節をイメージした
配色に装飾にエフェクト、という具合で派手に仕上がってます。
後半のサビ前からは見どころです。

全てが合わさり、とてもイイ作品が出来上がりました！
楽しく遊んでいただければと思います。

最後に、
BOF2011チームメンバーのfanfanさん、五条下位さん、むらしんさん、カギさん
チームの各楽曲にてボーカルや楽器でサポートいただいたpicoさん、PHEVOTTさん、仲村芽衣子さん、BBさん
BOF2011はちょっとがんばりたいぞー！と思い立った時に相談に乗ってくれたTOHRU MiTSUHASHiさん
テストプレイと難易度確認してくださったyujurieさん、Xetthoさん、kei iwataさん
その他途中段階の曲を聴いたりムービー見て意見くださった皆様
ありがとうございました！！！


[ COMMENT from cranky ]
よく考えたらBOFに本名義で参加するのは初めてな気がします。
モロにsiromaru + crankyって感じの曲が出来上がりました。狙い通りです。
作品の中では全体の構成とコード及びメロディを担当しています。
あ、あと音切り。もうピアノの音切りは勘弁・・・

ステキな映像と相まって、独自の雰囲気のある作品が作れたと自負しています。
picoにもあれこれ注文しながらコーラスを歌ってもらいました。ありがとう。

こういった曲は好きな人は好きでしょうし、苦手な人は苦手でしょう。
しかし、一人でも多くの方に楽しんでプレイして頂けるととても嬉しく思います。
どうぞ宜しくお願いします。
